# PHP Box Designer
Boxes. There are Boxes. Everywhere.  

Simple app that draws boxes using chars with or without text content inside.

## Installation

- Clone this repo:
- `$ git clone https://github.com/terremoth/php-box-designer.git`
- `$ cd php-box-designer`
- `$ composer install`
